JRSL

Beautiful terminal-based presentations in Markdown

View on GitHub Get Started

Features

Markdown Slides

Write your presentations in familiar Markdown syntax with YAML metadata for slide configuration.

ASCII Art Titles

Automatic figlet-powered ASCII art titles make your slides stand out in any terminal.

Image Support

Display images using ASCII art or the Kitty graphics protocol for high-quality rendering.

Flexible Layouts

Position images top, center, or bottom. Side-by-side layouts for image and text combinations.

Theme Support

Base16 color themes let you match your presentation to your terminal and personal style.

Keyboard Navigation

Simple arrow key navigation with scroll support for long content. Press q to quit.

Getting Started

Install and run JRSL in seconds:

# Clone the repository
git clone https://github.com/ralsina/jrsl.git
cd jrsl

# Install dependencies and build
shards install
shards build

# Run your first presentation
./bin/jrsl charla/jrsl-demo.md

Usage

jrsl [-t <theme>] [--kitty] [<file>]

Options:
  -t <theme>      Color theme to use (base16 themes)
  --kitty         Use Kitty graphics protocol for images
  --list-themes   List available color themes

Navigation:
  Left/Right      Previous/Next slide
  Up/Down         Scroll long content
  q               Quit

Slide Example

title: My Slide
